Users could create a colofor object with correct property to use the color palette they want.
Correspondingly, for the elements they want to use the library, they should set these element's class as "colorfor base(base1/base2)/main(main1/main2)/accent(accent1/accent2)" to tell the library what elements they want to apply library to and what kind color they want. For example, base means this element is like the base of the website, usually it can be light colors. Main means this element has basic content inside and their background color shoud be the theme color of the website. Accent means this element is something that you want to emphasize.

    Simple Steps:
  • Import the library with script tag
  • Choose element by using correct class
  • Create Colorfor instance in your js file which is used to manipulate the Dom
  • The slected elements will be colored nicely without set the color seperately

Code

             //JS
             var myColorforInstance = new Colorfor({ 
                elementClass: '.colorfor',
                states : {
                    "color": {
                        theme:'Blue',
                        type: 'vintage',
                        gradients: {
                            direction: '45deg'
                        },
                        animation:{
                            time: 10,         
                        },
                        gradientPic:{
                            blendingMode:"overlay"
                        },
                        selection:{
                            selectionColor:"accent"
                        },        
                    }
                }
                });
            
            //HTML
            <body class="colorfor base"></body>
            <div class="colorfor main gradient animation" id="bannerPic"> </div>
            <img class="colorfor gradientPic" src="img/bannerbanner.jpg" alt="banner Pic" />
            <div class="colorfor accent1"> </div>
            <div class="colorfor main" id="profileInfo"></div>
            <div class="colorfor main" id='timeline'></div>
            <button class="colorfor gradientButton" id="back">Back</button>

Color the elements with customizable property

Dipay some interesting elements use the lib

Color gradients with animation

Code

                //JS 
                var myColorforInstance = new Colorfor({ 
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            theme:'Orange',
                            type: 'vivid',
                            gradients: {           //must set the gradients to get gradients
                                direction: '45deg' //must set the gradient direction 
                            },
                            animation:{            //must set the animation to get the animation
                                time: 10,          //must set the animation speed  
                            }     
                        }
                    }
                    });


                //HTML
                <div class="colorfor main gradient animation"></div> 
                //set class like example with the same order

Color the page just by an image

User could just choose an image and library will produce color scheme according to this image and paint colors to the page

Click on image to see the page using color scheme produced by the image

Code

                //JS 
                var myColorforInstance = new Colorfor({
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            themePic:{               //must set the themePic to let lib know you choose to use the color produced by target image
                                gradients: {
                                    direction:'45deg'
                                },
                                animation:{
                                    time: 2,              
                                },
                                selection:{
                                    selectionColor:"accent"
                                }             
                            }
                
                        }
                    }
                    });

                //HTML
                <img class="colorfor themePic" src="img/bannerPic.jpg" alt="banner Pic" />
                //User has to set the image with class="colorfor themePic" to choose the select image
    Last thing to say:
  • The color scheme is not only three as shown before, so far we have 6 themes each has three modes(vintage/modern/vivid).
  • Users don't have to create instance for every single element, just create one with corresponding html elements. The js code shows before is just the essential property you have to set if you want to get corresponding functionality. That's to say, if you want more effects, you just need to add propery in the Colorfor instance and set related html elements.
  • The library uses the Vibrant.js to extract color from the image.
