Getting Started

Let's get started using Colorfor.js!

First, we need to include Colorfor.js in our page

                <script src='js/colorfor.js'></script>

Now, we need to create element which we want to use the library.

                //add colorfor to the class of the element to let the lib know which element you want to use lib
                //The second class should be the base/main/accent if this element is supposed to be painted color
                <body class="colorfor base">
                <div class="colorfor main gradient animation" > </div>
                <img class="colorfor gradientPic" src="img/bannerbanner.jpg" />
                <div class="colorfor accent1"> </div>
                <div class="colorfor main" ></div>
                <div class="colorfor main" ></div>
                <button class="colorfor gradientButton" >Back</button>
                </body>

Now, we can create a Colorfor object in the js file which manipulates the dom.

                    //JS 
                    var myColorforInstance = new Colorfor({ 
                        elementClass: '.colorfor', //Elements you want to use the lib should belong to class 'colorfor'
                        states : {
                            "color": {
                                theme:'Blue', //Must select the theme color
                                type: 'vintage', //Must select the color type(vintage/modern/vivid)
                                gradients: {    //Add gradients property if you want color gradients
                                    direction: '45deg' //Must choose direction of the gradients(to right/to top/45deg...)
                                },
                                animation:{ //Add gradients property if you want color gradients
                                    time: 10, //Must set time of the animation to control the animation speed
                                }
                            }
                        }
                        });

Configuration

The configuration is used to change how the library behaves. The configuration will be explained with use cases.

There are three main modes:Built-in color, ThemePic and RandomColor

Built-in color Mode

                    //JS
                    var myColorforInstance = new Colorfor({ 
                       elementClass: '.colorfor', //Elements you want to use the lib should belong to class 'colorfor'
                       states : {
                           "color": {
                               theme:'Blue', //Must select the theme color
                               type: 'vintage', //Must select the color type(vintage/modern/vivid)
                               gradients: {    //Add gradients property if you want color gradients
                                   direction: '45deg' //Must choose direction of the gradients(to right/to top/45deg...)
                               },
                               animation:{ //Add gradients property if you want color gradients
                                   time: 10, //Must set time of the animation to control the animation speed
                               },
                               gradientPic:{  //Add property gradientPic if you want pic blend with gradient color
                                   blendingMode:"overlay" //Must set blendingMode (soft-light/overlay/difference...)
                               },
                               selection:{   //Add property selection if you want selection with color
                                   selectionColor:"accent"  //Choose the color in the scheme set the selection color
                               },        
                           }
                       }
                       });
                
                   
                   //HTML
                   //add colorfor to the class of the element to let the lib know which element you want to use lib
                   //The second class should be the base/main/accent if this element is supposed to be painted color
                   <body class="colorfor base"></body>
                   <div class="colorfor main gradient animation" id="bannerPic"> </div>
                   <img class="colorfor gradientPic" src="img/bannerbanner.jpg" alt="banner Pic" />
                   <div class="colorfor accent1"> </div>
                   <div class="colorfor main" id="profileInfo"></div>
                   <div class="colorfor main" id='timeline'></div>
                   <button class="colorfor gradientButton" id="back">Back</button>

    Requirements:

  • Must set theme and type

    Parameters:

  • theme: Blue, Green, Pink, Coffee, Orange, Purple

  • type: vintage, modern, vivid


ThemePic mode

                //JS 
                var myColorforInstance = new Colorfor({
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            themePic:{               //must set the themePic to let lib know you choose to use the color produced by target image
                                gradients: {
                                    direction:'45deg'
                                },
                                animation:{
                                    time: 2,              
                                },
                                selection:{
                                    selectionColor:"accent"
                                }             
                            }
                
                        }
                    }
                    });

                //HTML>
                <img class="colorfor themePic" src="img/bannerPic.jpg" alt="banner Pic" />
                //User has to set the image with class="colorfor themePic" to choose the select image
                //Set other corresponding element like we do in built-in color mode

    Requirements:

  • Must set themePic

  • Must create an image element with class="colorfor themePic"

                //HTML

Color gradients with animation

                //JS 
                //All three modes can implement the gradients with animation, but users must set property gradients and animation
                var myColorforInstance = new Colorfor({ 
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            theme:'Orange',
                            type: 'vivid',
                            gradients: {           //must set the gradients to get gradients
                                direction: '45deg' //must set the gradient direction 
                            },
                            animation:{            //must set the animation to get the animation
                                time: 40,          //must set the animation speed  
                            }     
                        }
                    }
                    });


                //HTML
                <div class="colorfor main gradient animation"></div> 
                //set class like example with the same order

    Requirements:

  • Must set gradients and animation

  • Must set gradient and animation in the class of the corresponding html element

    Parameters:

  • direction: to right, to left, to bottom right, to top right, to bottom left, to top left, any angle like 45deg, 70deg

  • time: choose a int number to control the animation speed

Blend picture with color gradients and animation

                //JS 
                //All three modes can implement blending picture with color gradients and animation, but users must set property gradients and animation
                //Users must set the property gradientPic with its blendingMode
                var myColorforInstance = new Colorfor({ 
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            theme:'Orange',
                            type: 'vivid',
                            gradients: {           //must set the gradients to get gradients
                                direction: '45deg' //must set the gradient direction 
                            },
                            animation:{            //must set the animation to get the animation
                                time: 10,          //must set the animation speed  
                            },
                            gradientPic:{
                                blendingMode:"overlay" //must set the blend mode to get blending, (overlay/soft-light/saturation...)
                            }, 
                        }
                    }
                    });


                //HTML
                <div class="colorfor main gradient animation"> <img class="colorfor gradientPic" src="img/bannerbanner.jpg" alt="banner Pic" /></div>
                //User has to set the pic with class="colorfor gradientPic" to get the blend pic
                //User has to put the pic inside a colorfor element with gradient/animation color and make their sizes same.

    Requirements:

  • Must set gradients and animation

  • Must set gradientPic with blendingMode

  • Must set gradient and animation in the class of the corresponding html element

  • Must set the pic with class="colorfor gradientPic" to get the blend pic

  • Must put the pic inside a colorfor element with gradient/animation color and make their sizes same

    Parameters:

  • blendingMode: multiply, overlay, saturation, soft-light, difference, luminosity, lighten, darken

                var myColorforInstance = new Colorfor({
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            theme:'Orange',
                            type: 'vivid',
                            selection:{                 
                                selectionColor:"accent" 
                            },

                    }
                    });

                //HTML

                //User has to set the text with class="colorfor animatedText" to get animated text

  • Must set the value for the attribute data-text, it should be the same with the content of the animated text

Colorful and cute decorations

                //JS
                //You must make sure you have use a color scheme for the page, because the decoration color refers to the color scheme. You could use the built in sheme or choose themepic mode or random color mode.
                //If you choose the themepic mode or random color mode, the library will use the color automatically. Example below is using the built in color scheme
                var myColorforInstance = new Colorfor({
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            theme:'Orange',
                            type: 'vivid',
                            gradients: {
                                direction: '45deg'
                            },
                            animation:{
                                time: 10,         
                            },
                            gradientPic:{
                                blendingMode:"overlay"
                            },
                            selection:{
                                selectionColor:"accent"
                            },
                          
                            colorforDecoration:{   //User must add this property to have the decoration and set the shape property
                                shape: "star"      //Must choose between circle, heart and star
                            },
                
                        }
                    }
                    });
                

                //HTML
                //User does not have to create html element, the decoration elements will be created by the library.

    Requirements:

  • Must make sure you have use a color scheme for the page

  • Must set colorforDecoration with shape property

    Parameters:

  • shape: circle, heart, star


We recommand you use the library in the function so that any variables and code that need to be run will not pollute the global namespace or conflict with other functions or libraries

Code

                function useColorfor() {
                    var myColorforInstance = new Colorfor({
                        elementClass: '.colorfor',
                        states : {
                            "color": {
                                theme:'Orange',
                                type: 'vivid',
                                gradients: {
                                    direction: '45deg'
                                },
                                animation:{
                                    time: 40,         
                                },
                                gradientPic:{
                                    blendingMode:"overlay"
                                },
                                selection:{
                                    selectionColor:"accent"
                                },
                                colorforDecoration:{
                                    shape: "star"
                                },
                    
                            }
                        }
                        });
                    }
                    useColorfor();
