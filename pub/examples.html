Introduction

What is colorfor.js?

Colorfor.js is a library providing color schemes when you are worried about the colors of your web app. It creates dozens of pretty color schemes for you. It can not only provide color schemes, but also helps you apply these colors directly to your web page. It will produce elegant color combinations of the components to make simple web app with vivd colors and animations if users have chosen which elements to be painted by creaing certain elements with corresponding class.

    There are three main modes:

  • Built-in color: use built-in color scheme.

  • ThemePic: chose an image as theme picure, the library will produce the color scheme based on the image and apply the color to the page.

  • RandomColor: the library will generate color randomly and apply the color to the page.

Why colorfor.js?

If you are a beginner of web developer who wants to focus on the functionality fo your web and don't know how to pick color for your webpages, use colorfor.js to construct your web page quickly.

If you are building simple web pages, like a demo web page in a teaching class,colorfor.js can help you to quickly build a demo app with a bit of UI to make this demo more elegant will be so convenient.

If you are under these circumstances mentioned before, don't hesitate use Colorfor.js. Save lots of time!

Users could create a colofor object with correct property to use the built-in color palette they want.
Correspondingly, for the elements they want to use the library, they should set these element's class as "colorfor base(base1/base2)/main(main1/main2)/accent(accent1/accent2)" to tell the library what elements they want to apply library to and what kind of color they want. For example, base means this element is like the base of the website, usually it can be light colors. Main means this element has basic content inside and their background color shoud be the theme color of the website. Accent means this element is something that you want to emphasize.

Code (take vintage blue for example)

             //JS
             var myColorforInstance = new Colorfor({ 
                elementClass: '.colorfor', //Elements you want to use the lib should belong to class 'colorfor'
                states : {
                    "color": {
                        theme:'Blue', //Must select the theme color
                        type: 'vintage', //Must select the color type(vintage/modern/vivid)
                        gradients: {    //Add gradients property if you want color gradients
                            direction: '45deg' //Must choose direction of the gradients(to right/to top/45deg...)
                        },
                        animation:{ //Add gradients property if you want color gradients
                            time: 10, //Must set time of the animation to control the animation speed
                        },
                        gradientPic:{  //Add property gradientPic if you want pic blend with gradient color
                            blendingMode:"overlay" //Must set blendingMode (soft-light/overlay/difference...)
                        },
                        selection:{   //Add property selection if you want selection with color
                            selectionColor:"accent"  //Choose the color in the scheme set the selection color
                        },        
                    }
                }
                });
         
            
            //HTML
            //add colorfor to the class of the element to let the lib know which element you want to use lib
            //The second class should be the base/main/accent if this element is supposed to be painted color
            <body class="colorfor base"></body>
            <div class="colorfor main gradient animation" id="bannerPic"> </div>
            <img class="colorfor gradientPic" src="img/bannerbanner.jpg" alt="banner Pic" />
            <div class="colorfor accent1"> </div>
            <div class="colorfor main" id="profileInfo"></div>
            <div class="colorfor main" id='timeline'></div>
            <button class="colorfor gradientButton" id="back">Back</button>

ThemePic: Color the page just by an image

Users could just choose an image and library will produce color scheme according to this image and paint colors to the page

The library produce colors randomly and has an algorithm to calculate the brightness of the color to assign different colors to be base, main or accent.

Click on image to see the page using color scheme produced by this image

Code

                //JS 
                var myColorforInstance = new Colorfor({
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            themePic:{               //must set the themePic to let lib know you choose to use the color produced by target image
                                gradients: {
                                    direction:'45deg'
                                },
                                animation:{
                                    time: 2,              
                                },
                                selection:{
                                    selectionColor:"accent"
                                }             
                            }
                
                        }
                    }
                    });

                //HTML>
                <img class="colorfor themePic" src="img/bannerPic.jpg" alt="banner Pic" />
                //User has to set the image with class="colorfor themePic" to choose the select image
                //Set other corresponding element like we do in built-in color mode

RandomColor: Color the page just by random colors

Users could just choose the random color mode to paint the page by random color palette. The parameters set is similar the themePic mode.

The library produce colors randomly and has an algorithm to calculate the brightness of the color to assign different colors to be base, main or accent.

Click on the text: [Random Color] to see the page using random color scheme

If you refresh that page the random color scheme will keep changing

Random Color

Animated Text

Library could make animation effects of gradually appearing. We recommend you to give the animated texts a darker background to make the effect nicer. The animation only play once, you could refresh the page to see the effect multiple times.

Animated Text

Code

                //JS 
                //You don't need to set specific property in the object, but you must make sure you have set the choose a color scheme to get the gradient effect
                //If you choose the themePic mode or randomColor mode, the library will use the color automatically. Example below is using the built in color scheme
                
                var myColorforInstance = new Colorfor({
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            theme:'Orange',
                            type: 'vivid',
                            selection:{                 
                                selectionColor:"accent" 
                            },             
                        }
                    }
                    });

                //HTML
                <h1 class="colorfor animatedText" id= "exampleText" data-text="Animated Text">Animated Text</h1>
                //User has to set the text with class="colorfor animatedText" to get animated text
                //Another import tip is to set the value for the attribute data-text, it should be the same with the content of the animated text

Glowing Border

Library could make glowing border for the target block. The color scheme for the glowing border is random, which means that the glowing border will change every time you refresh the page. You could refresh the page to see the different glowing borders.

The element with glowing border should be the direct children of the body, or it will not show because of the z-index Property.

Colorful and cute decorations

The library could make animated and floating decorations floating on the page by just clicking the page. The decoration will be generated wherever and whenever you click on the page and it will fly like a balloon. Each decoration's color is come from the built in color scheme randomly or it will be from the themePic color scheme or randomColor scheme randomly.

In this page, the decoration shape is star. You could click anywhere to see the effect.

We have three different shapes decorations: circle, heart and star. Click on the corresponding shape below to see the pages with different decorations.

Now, in this page, you could click many times and many places to see the sky full of stars!

Code

                //JS
                //You must make sure you have use a color scheme for the page, because the decoration color refers to the color scheme. You could use the built in sheme or choose themepic mode or random color mode.
                //If you choose the themepic mode or random color mode, the library will use the color automatically. Example below is using the built in color scheme
                var myColorforInstance = new Colorfor({
                    elementClass: '.colorfor',
                    states : {
                        "color": {
                            theme:'Orange',
                            type: 'vivid',
                            gradients: {
                                direction: '45deg'
                            },
                            animation:{
                                time: 10,         
                            },
                            gradientPic:{
                                blendingMode:"overlay"
                            },
                            selection:{
                                selectionColor:"accent"
                            },
                          
                            colorforDecoration:{   //User must add this property to have the decoration and set the shape property
                                shape: "star"      //Must choose between circle, heart and star
                            },
                
                        }
                    }
                    });
                

                //HTML
                //User does not have to create html element, the decoration elements will be created by the library.

  • Users don't have to create instance for every single element, just create one with corresponding html elements. That's to say, if you want more effects, you just need to add propery in the Colorfor instance and set related html elements.
